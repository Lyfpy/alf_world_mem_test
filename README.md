# AlfWorld Memory

## Children Overview
| File | When to Use |
|------|-------------|
| tasks/two-object-placement.md | Use when the task involves picking up two identical objects and placing them in the same destination location. |
| tasks/object-examination.md | Use for tasks involving examining or looking at objects, especially when there are challenges or specific spatial relationships mentioned. |
| tasks/object-cooling.md | Use this memory when tasked with cooling objects (food items or containers) using the fridge and then placing them at a specified destination location. |
| tasks/food-cleaning.md | Use this memory when the task involves cleaning food items (typically vegetables) using a sinkbasin. This pattern applies when you need to locate a food item, clean it at the sinkbasin, and then place it at a specified destination. |
| tasks/food-heating.md | Use for tasks that involve heating food items (typically bread or similar items) using microwave before placing them at a destination. |
| tasks/simple-placement.md | For tasks that involve moving a single object from its current location to a destination without any intermediate processing steps |
